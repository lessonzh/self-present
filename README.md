# 作品管理系统

基于nodeJs、koa2、mongodb的作品管理系统

只是为了想把自己的一些项目放到线上展示，所以进行了简单开发，用时半天

后端服务使用nodeJS框架express、数据库使用在线免费的mongodb数据库mlab

---

## 实现功能
- 项目添加
- 项目删除
- 项目修改
- 项目排序

## 主要技术栈列表
- Node
- express
- Mongoose
- MongoDB
